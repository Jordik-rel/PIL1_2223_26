Objectifs du projet

Le projet de cette année consiste à réaliser une application web de gestion des emplois du temps pour un établissement d'enseignement supérieur.Il s'agira entre autres de:

  • Développer une interface conviviale permettant aux utilisateurs de consulter et de gérer leurs emplois du temps de manière efficace.
  • Mettre en place un système de gestion des utilisateurs, avec des fonctionnalités d'inscription, de connexion et de gestion des profils.

    Instructions de Déploiement

    Pour déployer le projet, suivez les étapes suivantes :

    1. Accédez à la page GitHub du dépôt contenant les codes de l'application web.
    2. Cliquez sur le bouton "Code" pour télécharger une copie du code source de l'application. Sélectionnez "Download ZIP" pour télécharger une version compressée du dépôt.
    3. Extrayez le contenu du fichier ZIP téléchargé dans un emplacement de votre choix sur votre ordinateur.
    4. Recherchez un fichier d'entrée principal de l'application (par exemple, index.html ou app.py) ou tout autre fichier qui semble être le point de départ de l'application.
    5. Double-cliquez sur le fichier d'entrée principal pour lancer l'application dans votre navigateur par défaut. Dans certains cas, vous devrez peut-être exécuter un fichier exécutable ou utiliser une ligne de commande spécifique pour lancer l'application.
    6. L'application devrait maintenant s'ouvrir dans votre navigateur et vous pourrez l'utiliser.

    Conception de l'application

    L'application de gestion des emplois du temps a été conçue en utilisant différentes techniques pour garantir une interface conviviale et une gestion efficace des données.

    L'interface web de l'application est basée sur une structure HTML simple et bien organisée. Chaque page est composée de différentes sections pour afficher les informations essentielles. Par exemple, la page d'accueil présente un aperçu des emplois du temps des différentes filières sous forme de tableau. Les autres fonctionnalités sont également présentées dans des sections dédiées pour une navigation intuitive.

    Le style CSS a été utilisé pour rendre l'interface de l'application attrayante et cohérente. Des couleurs appropriées, des polices lisibles et des mises en page bien définies ont été appliquées pour améliorer l'expérience utilisateur. Des styles responsives ont également été implémentés pour assurer une compatibilité avec différents appareils et tailles d'écran.

    La base de données a été conçue en utilisant le système de gestion de bases de données MySQL. Différentes entités ont été identifiées, telles que "cours", "enseignants", "filières", "salles de classe", etc. Chaque entité a été représentée par une table dans la base de données, avec des attributs appropriés pour stocker les informations spécifiques à chaque entité.

    Les relations entre les différentes entités ont été établies à l'aide de clés primaires et clés étrangères. Par exemple, la table des "cours" est liée à la table des "enseignants" par une clé étrangère qui représente l'enseignant responsable de chaque cours. De même, une relation entre les "cours" et les "salles de classe" est établie pour indiquer la salle où chaque cours a lieu.

    Manuel d'utilisation

      Accéder au site web :

    • Ouvrez votre navigateur web préféré.
    • Tapez l'URL du site web de l'application de gestion des emplois du temps dans la barre d'adresse.
    • Appuyez sur Entrée pour accéder au site.

      • Consulter les emplois du temps :

    • Sur la page d'accueil de l'application, recherchez une section ou un lien spécifique dédié aux emplois du temps.
    • Cliquez sur ce lien ou accédez à la section correspondante.
    • Sélectionnez les options appropriées, telles que la date, le département ou le groupe d'étudiants, pour afficher les emplois du temps souhaités.
    • Les emplois du temps correspondants s'afficheront à l'écran.
      • Créer un emploi du temps :
    • Identifiez la fonctionnalité de création d'un emploi du temps sur le site web.
    • Cliquez sur le bouton ou le lien indiquant la création d'un nouvel emploi du temps.
    • Remplissez les détails requis, tels que la date, l'heure, les cours, les enseignants et les salles de classe concernés.
    • Validez les informations saisies et enregistrez l'emploi du temps nouvellement créé.
    • Vous devriez voir une confirmation que l'emploi du temps a été créé avec succès.
      • Inscription sur le site :
    • Recherchez un lien ou une section dédiée à l'inscription sur le site web.
    • Cliquez sur ce lien pour accéder à la page d'inscription.
    • Remplissez les informations demandées, telles que le nom, l'adresse e-mail et le mot de passe.
    • Validez les informations et soumettez le formulaire d'inscription.
    • Vous devriez recevoir une confirmation d'inscription et pouvoir accéder aux fonctionnalités réservées aux utilisateurs inscrits.
