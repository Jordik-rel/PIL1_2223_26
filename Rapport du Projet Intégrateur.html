Instructions de Déploiement

Pour déployer le projet, suivez les étapes suivantes :

  • Manuel d'utilisation

      Accéder au site web :

    • Ouvrez votre navigateur web préféré. Tapez l'URL du site web de l'application de gestion des emplois du temps dans la barre d'adresse. Appuyez sur Entrée pour accéder au site.

      • Consulter les emplois du temps :

    • Sur la page d'accueil de l'application, recherchez une section ou un lien spécifique dédié aux emplois du temps. Cliquez sur ce lien ou accédez à la section correspondante. Sélectionnez les options appropriées, telles que la date, le département ou le groupe d'étudiants, pour afficher les emplois du temps souhaités. Les emplois du temps correspondants s'afficheront à l'écran.
      • Créer un emploi du temps :
    • Identifiez la fonctionnalité de création d'un emploi du temps sur le site web. Cliquez sur le bouton ou le lien indiquant la création d'un nouvel emploi du temps. Remplissez les détails requis, tels que la date, l'heure, les cours, les enseignants et les salles de classe concernés. Validez les informations saisies et enregistrez l'emploi du temps nouvellement créé. Vous devriez voir une confirmation que l'emploi du temps a été créé avec succès.
      • Inscription sur le site :
    • Recherchez un lien ou une section dédiée à l'inscription sur le site web. Cliquez sur ce lien pour accéder à la page d'inscription. Remplissez les informations demandées, telles que le nom, l'adresse e-mail et le mot de passe. Validez les informations et soumettez le formulaire d'inscription. Vous devriez recevoir une confirmation d'inscription et pouvoir accéder aux fonctionnalités réservées aux utilisateurs inscrits.
